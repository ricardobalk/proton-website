# ProtonMail website

This is a replica of ProtonMail's website (protonmail.com), made in [Next.js][Next.js Homepage], the React Framework for Production.

## Getting started

There will be a Docker setup in a short while. For now &mdash;Â Make sure that you have a proper installation of Node (12.x+), have set up the correct environment variables, etc.

**Installing the dependencies**

```sh
yarn
# or: npm install
```

**Running a development server**

```sh
yarn dev
# or: npm run dev
```

**Building the website**

```sh
yarn build
# or: npm run build
```
